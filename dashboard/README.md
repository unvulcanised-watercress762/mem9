# Dashboard

This directory is the dedicated home for mem9 dashboard work.

## Structure

- `docs/` — product specs, UX notes, and related dashboard documents
- `app/` — dashboard application code and assets

The goal is to keep dashboard planning and implementation isolated from the main marketing site and general project docs.

## Docs

- `docs/dashboard-mvp-spec.md` — product spec
- `docs/information-architecture.md` — information architecture (two-page design)
- `docs/data-contract.md` — API and backend alignment
- `docs/dev-tasks.md` — development task breakdown
